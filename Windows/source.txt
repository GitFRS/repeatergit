Для создания раздела введите MKDIR
Для удаления раздела введите DELDIR
Для редактирования раздела введите EDITDIR
Для вывода содержимого раздела введите LISTDIR <имя_раздела>
Для вывода всего содержимого в списке введите LISTALL
Для вывода подсказок введите HELP
Для выхода введите EXIT
